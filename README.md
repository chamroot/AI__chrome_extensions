## X Auto Refresh

X（旧Twitter）のタイムラインを自動更新するChrome拡張機能です。

### 主な機能

* タイムラインの自動更新
* 更新間隔の設定
* 曜日・時間帯による自動更新スケジュール
* 未読ポストの管理・ハイライト
* キーワードハイライト
* ユーザーハイライト
* コンソールログのON/OFF設定
