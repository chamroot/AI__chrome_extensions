## X Auto Refresh

X（旧Twitter）のタイムラインを自動更新するChrome拡張機能です。

### 主な機能

* タイムラインの自動更新
* 更新間隔の設定
* 曜日・時間帯による自動更新スケジュール
* 未読ポストの管理・ハイライト
* キーワードハイライト
* ユーザーハイライト
* コンソールログのON/OFF設定

## Audio Controller for YouTube

YouTube動画の音量・再生速度・音質（EQ）を直感的にコントロールできるChrome拡張機能です。

### 主な機能

* 音量拡張（0〜200%）
* 再生速度調整（0.25〜10.00倍）
* 3Band イコライザー（LOW(250Hz以下) / MID(1000Hz付近) / HIGH(4000Hz以上)）
* オーディオモニター（RMSレベルのリアルタイム表示）
* 一括リセット機能

### 注意事項

* 再生速度を調整する他の拡張機能を入れている場合、正常に作動しない場合があります。
